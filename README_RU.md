# PodsGuard 1.1

Бесплатная защита от пауз AirPods на Android.

## Что изменено в 1.1

- исправлена главная проблема версии 1.0: теперь проверка не зависит только от одного перехода `PLAYING → PAUSED`;
- добавлен быстрый watchdog примерно каждые 140 мс;
- добавлена постоянная фоновая служба с уведомлением;
- добавлен повторный `MediaController.play()`;
- добавлен запасной системный `KEYCODE_MEDIA_PLAY`, если плеер проигнорировал первую команду;
- распознаются `PAUSED`, `STOPPED`, `NONE` и `ERROR` после недавнего воспроизведения;
- добавлено восстановление после перезагрузки телефона;
- добавлена кнопка настройки фоновой работы для iQOO/vivo.

## Важно

PodsGuard не меняет внутреннюю настройку AirPods «Автообнаружение уха». Обычный Android‑APK не имеет прямого доступа к фирменному протоколу Apple. Приложение максимально быстро возвращает воспроизведение после команды паузы.

Чтобы паузы не было вообще даже на долю секунды, выключите «Автообнаружение уха» через iPhone/iPad/Mac или используйте решение с доступом к фирменному протоколу AirPods (например LibrePods на совместимой системе/root).

## После установки

1. Откройте PodsGuard.
2. Выдайте доступ к уведомлениям/медиасессиям.
3. Разрешите уведомления приложения.
4. Включите защиту.
5. Для iQOO/vivo откройте «Настройки приложения» и выставьте батарею «Без ограничений»; разрешите автозапуск, если такой пункт есть.
6. Поставьте режим «Мгновенно — около 80 мс».
